### Fablo REST

A simple REST API interface for Hyperledger Fabric blockchain network.
It is distributed as Docker image: `softwaremill/fablo-rest`.

## Endpoints

`POST /user/enroll`
* Headers: 

`/user/reenroll`

`/user/reenroll`

`/user/register`

`/discover`

`/invoke/:channel/:chaincode`
